## Argilla Instance


### Quick Start

#### Spin Up/Down

```zsh
docker compose up
```

Spin down:

```zsh
docker compose down
```


#### Extract Annotated Sets

Dump json:

```zsh
python dump_argilla_data --dataset='your-dataset-name' --workspace='your-workspace-name' --outpath='./path/to/your/desired/outfile.json'
```
